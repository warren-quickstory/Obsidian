# 上涨趋势波段接力系统 (T1)

**版本**: v1.1  
**状态**: 已定义（待沙盘推演）  
**创建日期**: 2026-04-09  
**最后更新**: 2026-04-09  

## 系统概述
适用于A股市场，在已证明的多头趋势中，捕捉波段回踩后的二次启动机会。核心哲学是“让利润奔跑，让亏损截断”。

## 核心文件
- `README.md` - 本文件，策略总览
- `system_technical.md` - 完整技术文档（结构化描述）
- `params.json` - 所有可调参数（风险、阈值等）
- `tdx_formula.txt` - 通达信选股公式（环境过滤器）
- `changelog.md` - 版本更新历史

## 目录结构
- `backtest/` - 回测结果与日志
- `signals/` - 实战或沙盘信号记录

## 快速启动
1. 阅读 `system_technical.md` 了解完整逻辑。
2. 根据自身资金调整 `params.json` 中的 `base_risk`（基础风险额度）。
3. 将 `tdx_formula.txt` 导入通达信进行选股。
4. 执行沙盘推演，记录结果至 `signals/`。
