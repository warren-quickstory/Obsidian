## 2. 执行合规性审查（诚实面对自己）
*请打勾（[x] 或 [ ]）*
*   [ ] **有计划才交易**：这笔交易是否在盘前/开仓前写下了明确的计划？
*   [ ] **顺应大势**：是否顺应了日线/4小时级别的大方向？
*   [ ] **结构确立**：是否等待了市场给出明确的止损结构（次高/次低/横盘箱体）才进场？
*   [ ] **严格止损**：止损单是否在开仓时同步挂出？**绝对没有向下移动过止损位？**
*   [ ] **没有手痒**：入场点是否完全符合系统定义的形态（未受急涨急跌的FOMO情绪驱使追单）？

**执行力综合评分**：
*   [ ] **A级（完美）**：100%按计划和系统执行，无论盈亏。
*   [ ] **B级（及格）**：基本按计划，但在进出场时机上有轻微主观偏差。
*   [ ] **C级（不及格）**：无计划交易、冲动追单、或者提前获利平仓。
*   [ ] **D级（致命）**：扛单、逆势加仓、移动止损、重仓赌博。
