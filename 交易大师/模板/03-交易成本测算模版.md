# 💰 【系统风控】商品期货交易成本测算模版

> **导师戒律**：忽视摩擦成本的交易系统，如同漏水的木桶。短线和日内交易中，“平今惩罚”和“滑点”往往是盈亏比的隐形杀手。必须对你常做品种的手续费特性了如指掌。

## 一、 核心计算公式备忘录

商品期货手续费主要分为两种收取方式，开仓前必须确认：

### 1. 绝对值法（固定金额收费）
*   **公式**：`交易成本 = 手数 × 固定费率`
*   **计算极简**：不受当时合约价格影响。
*   *(例如：黄金 au 开仓 15元/手，大豆 y 3.75元/手)*

### 2. 比例费率法（按合约价值收费）
*   **公式**：`交易成本 = 建仓价格 × 合约乘数（吨/手） × 手数 × 比例费率`
*   **特点**：价格越高，手续费越贵！
*   *(例如：螺纹钢 rb 按万分之0.3收取；若在 3600元 建仓，1手(10吨)成本 = 3600 × 10 × 1 × 0.00003 = 1.08元)*

## 二、 重点关注品种费率特征卡 (结合华泰期货标准)
*(在使用时，请将你核心关注的3-5个品种填入下方)*

### 品种A：[ 例如：纯碱 SA ]
*   **计费方式**：比例类 —— [ 万分之 3 ] 
*   **合约乘数**：[ 20 ] 吨/手
*   **开仓手续费**：万 3
*   **平今手续费**：万 3 *(注：无平今优惠，日内短线成本较高)*
*   **日内单手成本估算**（假设2000点）：`2000 × 20 × 0.0003 × 2(一开一平) = 24元/手`

### 品种B：[ 例如：苹果 AP ]
*   **计费方式**：固定类
*   **合约乘数**：[ 10 ] 吨/手
*   **开仓手续费**：7.5 元/手
*   **平今手续费**：30 元/手 🚨**(剧烈惩罚！日内短线必须规避或大幅提高盈亏比预期)**
*   **日内单手成本估算**：`7.5 + 30 = 37.5元/手`

### 品种C：[ 例如：黄金 au ]
*   **计费方式**：固定类
*   **合约乘数**：[ 1000 ] 克/手
*   **开仓手续费**：15 元/手
*   **平今手续费**：0 元/手 🟢**(免平今！极度适合日内试错与短线操作)**
*   **日内单手成本估算**：`15 + 0 = 15元/手`

## 三、 单笔实战成本估算表 (计划开仓时填入)

| 项目              | 数据录入 / 计算结果                                |
| :-------------- | :----------------------------------------- |
| **交易品种及合约**     | (例：螺纹 2610)                                |
| **建仓点位**        | (例：3500)                                   |
| **止损点位 / 空间**   | (例：3480 / 止损空间 20点)                        |
| **目标点位 / 空间**   | (例：3560 / 盈利空间 60点)                        |
| **计划开仓手数**      | (例：10 手)                                   |
| **单手毛利润估算**     | (盈利空间 × 乘数) = X 元                          |
| **单手毛亏损估算**     | (止损空间 × 乘数) = Y 元                          |
| **表观盈亏比**       | (毛利润 / 毛亏损) = 3.0                          |
| **预计交易成本(开+平)** | (算入手续费与至少1跳滑点) = Z 元/手                     |
| **真实单手净亏损**     | (毛亏损 + 交易成本) = 真实底线                        |
| **真实单手净利润**     | (毛利润 - 交易成本) = 真实预期                        |
| **真实盈亏比 (关键)**  | **真实净利润 / 真实净亏损** *(注：若真实盈亏比跌破 1.5，放弃该交易)* |

## 导师实战忠告：关于“平今规则”的战术应用

1. **对于“平今免收”（如碳酸锂、黄金、部分外盘品种）**：
   * 这种品种极大降低了试错成本，非常适合使用你的**“1分钟动量降维打击法”**。一旦发现动能不足，可以直接微利或保本平仓，不用心疼手续费。
   
2. **对于“平今高昂”（如苹果AP、股指IC）**：
   * 这种品种**严禁随意做短线/1分钟级别的试错**！一开一平的成本太高，甚至能吃掉你一段完整的微观波段利润。
   * 应对策略：要么提升入场级别（改做1小时/日线波段，隔夜平仓避开平今费）；要么将微观止损空间拉大，减少被来回扫损的次数。

3. **永远把“滑点(Slippage)”算进成本**：
   * 除了明面的手续费，急涨急跌行情中你的市价单通常会有1-2跳（Tick）的滑点。计算真实盈亏比时，必须把双向滑点加在手续费里。
